## 💻 Sobre Mim

Olá! Sou a Rapha. Atualmente trabalho como **Assistente de Desenvolvimento na PicPay**, integrado ao time de **DevSecOps com foco em Red Team** na área de cibersegurança. Combinando o desenvolvimento de software com a mentalidade ofensiva de segurança, atuo na identificação de vulnerabilidades e automação de processos seguros.

Paralelamente, estou no segundo ano do **Ensino Médio Técnico no Instituto J&F**, especializando-me em **Desenvolvimento de Sistemas e Análise de Dados**.

- 🎯 **Foco atual:** DevSecOps, Segurança Ofensiva (Red Team) e Engenharia de Dados.
- 🛡️ **No dia a dia:** Automatizando scripts de segurança, analisando código e manipulando grandes volumes de dados de forma segura.
